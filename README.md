# TroopTrack 🌼

A full-stack web application to help troop leaders manage meetings, activities, and volunteers.

## Tech Stack
- React
- Node.js / Express
- MongoDB
- Open-Meteo API

## Features
- Create, edit, delete meetings
- Assign badges and notes
- Weather integration for planning
- Activity management
- Volunteer tracking

## How to Run

### Backend
cd trooptrack-server  
npm install  
npm run dev  

### Frontend
cd trooptrack-client  
npm install  
npm run dev
